# Control de Versiones

## Herramienta Utilizada

Para el desarrollo del proyecto se utilizó **Git** como sistema de control de versiones y **GitHub** como repositorio remoto para almacenar el código fuente y facilitar el trabajo colaborativo entre los integrantes del equipo.

## Tipo de Versionamiento

Se adoptó el **Versionado Semántico (SemVer)**, utilizando el formato:

MAJOR.MINOR.PATCH

Ejemplo:

v1.5.1

Donde:

- **MAJOR:** Cambios importantes que afectan la compatibilidad.
- **MINOR:** Incorporación de nuevas funcionalidades.
- **PATCH:** Corrección de errores y mejoras menores.
